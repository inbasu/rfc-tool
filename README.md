### RFC Tool

CLI утилита для поиска, скачивания и чтения RFC документов.

### About

RFC (Request for Comments) — документы с техническими спецификациями и стандартами интернета. Эта утилита позволяет удобно искать, скачивать и читать RFC прямо из терминала.

#### Команды/флаги

- `-f, --find <query>` — Поиск RFC по ключевому слову в названии
- `-l, --list` — Список уже скачанных/кэшированных файлов  
- `-c, --clear [N...]` — Удаление кэшированных RFC (все или указанные номера)
- `-h, --help` — Показать сообщение помощи

#### Примеры использования

- `rfc 3261` — Открывает RFC 3261 (скачивает если нет локальной копии)
- `rfc -f sip` — Поиск RFC файла содержащего в названии SIP
- `rfc -l` — Покажет все локальные RFC
- `rfc -c` — Удалит все скачанные файлы
- `rfc -c 3261` — Удалит только RFC 3261
- `rfc -c 3261 3262 3263` — Удалит несколько RFC

#### Установка:
```
chmod +x install.sh && ./install.sh
```
