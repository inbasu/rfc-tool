### Утилита для скачивания и чтения RFC в less

#### Использование
Команды/флаги:
    -f, --find <query>      Поиск RFC по ключевому слову в названии
    -l, --list              Список уже скачанных/кэшированных файлов
    -c, --clear             Удаление скачанных/кешированных файлов
    -h, --help              Показать сообзение помощи

Примеры использования:
    rfc 3261                Открывает RFC 3261 (Скачивает если нет локальной копии)
    rfc -f sip              Поиск RFC файла содержащего в названии SIP
    rfc -l                  Покажет все локальные RFC
    rfc -c                  Удалит все скачанные файлы

#### Установка
'''
sudo ln -s "$(pwd)/rfc.py" /usr/local/bin/rfc
'''
